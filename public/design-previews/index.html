OctoTusk · Table treatment

Four options for the Octopus table.

The live /dashboard table works, but the dark-slate header reads as Bloomberg against the new lighter chrome, neon column tints feel "trading screen" not "research desk," and 11px row padding makes 17 columns hard to scan. Here are four different fixes — same data, four philosophies.

A · Light Headers

Calm →

Kills the dark slate entirely. Header becomes part of the same light aesthetic as the rest of the dashboard. Group tints are text-color only — no background fills.

no dark slate · text-color tints · 1px column-group hairlines · 11px body padding
B · Editorial Dense

Refined →

Keeps the dark header — but graduates it from "Bloomberg" to "FT/Economist." Calmer green and amber tints. 14px body padding (was 11px) is the single biggest readability gain.

deeper warmer slate · calmer tints · 14px row padding · softer Alt 8 · cap-tag inline
C · Scannable

Functional →

Aesthetics unchanged. Pure scanning improvements: sticky COMPANY column on horizontal scroll, faint zebra, whole-row hover, decimal alignment, taller magnitude bars.

sticky company column · ultra-faint zebra · whole-row hover · 4px left-anchored bars
D · Clean Numbers Latest

Tufte →

No conditional background tint anywhere — pill text + magnitude bar carry the signal. 1Y & 2Y forward cells show absolute target ₹ price below the % upside. Default sort: 1Y descending — actionable opportunities at the top.

no upside tints · 1Y/2Y show ₹ targets · sorted by 1Y desc · refined dark header from B
How to compare: open each option, then horizontal-scroll the table to test C's sticky column, hover a row to test C's whole-row highlight, and compare row density between A (11px), B (14px), and C (12px). On D, look at the 1Y/2Y cells (now show ₹ target under the %), the rows ordered by 1Y descending, and the absence of any conditional background tint in the upside columns. Use the theme toggle (top-right) to test light + dark on each.

All four render the same 12 rows with the same data — visual diff is treatment-only.
